# 🦠 COVID-19 Global Data Analysis Project

## 📌 Project Overview

This project is a beginner-friendly Python Data Analysis project that analyzes a COVID-19 global dataset. It demonstrates how to generate, clean, analyze, and visualize data using Python.

The project includes both:

- Object-Oriented Programming (OOP) version
- Procedural Programming (POP) version

It also provides a dataset generator to create a realistic COVID-19 dataset automatically.

---

## 📂 Project Structure

```
COVID19-Data-Analysis/
│
├── covid19_global_data.csv      # Generated dataset
├── data_generate.py             # Dataset generator
├── main_oop.py                  # OOP implementation
├── main_pop.py                  # POP implementation
├── main.ipynb                   # Jupyter Notebook
├── requirements.txt             # Required libraries
└── README.md
```

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/covid19-data-analysis.git
```

Go to the project folder

```bash
cd covid19-data-analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

### Object-Oriented Version

```bash
python main_oop.py
```

### Procedural Version

```bash
python main_pop.py
```

### Jupyter Notebook

```bash
jupyter notebook main.ipynb
```
